# Synoptic

**Synoptic** — это интерактивное веб-приложение для отображения текущей погоды в выбранных пользователем городах. Приложение позволяет вводить неограниченное городов и отображает погодные карточки с детальной информацией о температуре, ветре, осадках и других метеоданных.

## Особенности

- Добавление неограниченного количества городов.
- Отображение погодных карточек с:
  - Температурой и ощущаемой температурой
  - Влажностью
  - Облачностью и осадками
  - Ветром и порывами
  - Давлением и UV индексом
  - Временем последнего обновления
- Динамическое создание карточек через JS.
- Возможность удаления городов и карточек.
- Адаптивный интерфейс и визуальные эффекты (ночная тема).

## Технологии

- HTML, CSS, JavaScript
- Fetch API для получения данных с внешнего погодного API
- Модульная структура JS с использованием ES6 модулей

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Force-in-Motion/Synoptic.git
   ```
2. Откройте `index.html` в браузере.
3. Введите названия городов и нажмите "Добавить", чтобы отобразить карточки с погодой.

## Структура проекта

```
src/
 ├─ js/
 │   ├─ api.js        # Работа с API погоды
 │   ├─ card.js       # Логика создания карточек погоды
 │   └─ index.js      # Главный скрипт управления интерфейсом
 ├─ css/
 │   └─ style.css     # Стили приложения
 └─ index.html        # Главная страница
```
